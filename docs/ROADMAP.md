# 路线图

这份 Roadmap 会随着真实使用情况持续调整。

## 近期计划

- 给 README 增加截图或 GIF
- 增加交互控件的 accessibility 检查
- 补充 Tabs、Menu、Input 等控件的键盘行为说明
- 将更多静态展示页示例转换为 React / Vue 组件
- 补充更贴近 Atlas EIDS 的企业业务场景模板

## 组件方向

- 将 React 组件整理成可发布的 package
- 将 Vue 3 组件整理成可发布的 package
- 增加 Storybook 或类似组件工作台
- 为受控输入和状态变体增加组件测试

## 设计系统方向

- 增加 Figma token mapping
- 扩展图表示例
- 补充 Light Mode / Dark Mode 对比度规则
- 增加适合企业 Dashboard 的 compact density 变体

## 发布方向

- 创建第一个 tagged release
- 如果项目从示例走向组件库，发布 npm packages
- 增加自动化 CI，检查 React 与 Vue 示例构建
