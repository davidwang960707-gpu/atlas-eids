# 开源检查清单

公开仓库前建议逐项检查。

## 仓库信息

- [ ] 确认仓库名和简介
- [ ] 增加项目截图或 GIF
- [ ] 确认 License 选择
- [ ] 移除 `.DS_Store` 等本地文件
- [ ] 检查 `skills/` 内容，确认适合公开维护
- [ ] 确认没有私有 URL、凭证、token 或客户数据

## 文档

- [ ] README 说明项目是什么
- [ ] README 说明如何运行展示页
- [ ] React 和 Vue 示例有安装/构建说明
- [ ] 有贡献指南
- [ ] 有安全策略
- [ ] 有变更日志

## 代码

- [ ] 静态展示页可以正常打开
- [ ] React 示例可以构建
- [ ] Vue 3 示例可以构建
- [ ] Dark / Light Theme 可以切换
- [ ] 导航锚点正常
- [ ] 未提交 `node_modules` 或 `dist`

## 社区

- [ ] Issue 模板已准备
- [ ] Pull Request 模板已准备
- [ ] 行为准则已准备
- [ ] 维护者联系方式清晰
