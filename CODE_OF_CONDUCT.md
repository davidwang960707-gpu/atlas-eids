# 行为准则

## 我们的承诺

我们希望 Atlas EIDS 成为一个尊重、建设性、对设计师和工程师都友好的开源项目。

## 鼓励的行为

- 反馈具体、友善、可执行。
- 默认善意，同时清楚指出问题。
- 评价作品和方案，不攻击个人。
- 尊重不同经验水平的参与者。
- 让讨论围绕项目本身展开。

## 不接受的行为

- 骚扰、辱骂、威胁或人身攻击
- 歧视性语言或行为
- 未经允许公开他人隐私信息
- 持续发布无关内容并干扰项目讨论

## 处理方式

维护者可以删除不当内容、关闭 issue，或限制破坏社区氛围的参与行为。

## 反馈渠道

如果你需要反馈社区行为问题，可以通过仓库维护者主页或 [SECURITY.md](SECURITY.md) 中的安全联系路径联系维护者。
