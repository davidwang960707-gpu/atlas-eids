01 · Philosophy

Spatial & Alive

摒弃传统 SaaS 的"后台感"与"表格堆砌"。Atlas EIDS 以"空间化工作台"为理念， 通过辉光、粒子与呼吸动效，赋予系统"正在苏醒的智能生命体"气质。

去后台化

减少边框与割裂感，让 AI 成为视觉与交互的绝对中心。

信息呼吸感

大量留白与毛玻璃材质，降低认知负载，提供沉浸式思考空间。

生命体微交互

流体、呼吸、漂浮、光晕。每一次 Hover 都是与 AI 意识的触碰。

07 · Code Examples

Implementation

提供 React 和 Vue 3 两个版本的组件实现，满足不同技术栈项目的需求。

Agent Orb 组件

// AgentOrb.tsx
import React from 'react';
import './AgentOrb.css';

type OrbState = 'idle' | 'thinking' | 'running' | 'error';

interface AgentOrbProps {
  state?: OrbState;
  size?: number;
}

const AgentOrb: React.FC<AgentOrbProps> = ({
  state = 'idle',
  size = 80
}) => {
  return (
    <div className={`orb-wrapper state-${state}`} style={{ width: size, height: size }}>
      <div className="orb-ring"></div>
      <div className="orb" style={{ width: size * 0.67, height: size * 0.67 }}></div>
    </div>
  );
};

export default AgentOrb;
<!-- AgentOrb.vue -->
<template>
  <div :class="['orb-wrapper', `state-${state}`]" :style="{ width: size + 'px', height: size + 'px' }">
    <div class="orb-ring"></div>
    <div class="orb" :style="{ width: size * 0.67 + 'px', height: size * 0.67 + 'px' }"></div>
  </div>
</template>

<script setup lang="ts">
type OrbState = 'idle' | 'thinking' | 'running' | 'error';

interface Props {
  state?: OrbState;
  size?: number;
}

withDefaults(defineProps<Props>(), {
  state: 'idle',
  size: 80
});
</script>
