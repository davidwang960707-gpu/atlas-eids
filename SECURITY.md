# 安全策略

## 支持版本

Atlas EIDS 目前处于 1.0 之前的阶段。安全修复默认会合入主分支，后续如创建 release branch，会再补充版本支持策略。

## 报告安全问题

请不要通过公开 issue 披露安全漏洞。

如果你发现安全问题，请私下联系仓库维护者，并尽量提供：

- 问题描述
- 复现步骤
- 受影响的文件或 package
- 可能影响
- 如果已有思路，可附上修复建议

## 范围

本项目主要是前端设计系统展示页和组件示例仓库。安全问题通常包括：

- 不安全的 JavaScript 行为
- 示例依赖中的安全漏洞
- Demo 中可能导致 XSS 的实现
- 文档中可能引导用户采用不安全做法的说明
